# API Clínica de Odontología

### API para gestionar aspectos de una clínica de odontología como administrar odontólogos, pacientes, turnos y datos personales de importancia.
---

## Documentación
### Para acceder a la interfaz de swagger del proyecto es necesario acceder a la siguiente ruta despues de iniciar la API: http://localhost:8080/swagger-ui/index.html

### Como base de datos se usa H2.

### Para interactuar con la API se creó un frontend muy sencillo como muestra que permite registrar nuevos odontólogos y visualizar todos los odontólogos existentes en una lista de tarjetas. El archivo del frontend se llama: `clinica-odontologica.html` y se encuentra en la carpeta raíz del repositorio.
